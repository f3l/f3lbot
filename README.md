# F3LBot #

A collection of ErrBot-Extensions made for F3L.

## Implemented ##

* Give: beer, beer for user, give, give to user, listen for "beer"
* Pkg:
  * aur: package search, package info, maintainer info
  * arch: package search, package info, maintainer info
  * pkg: package search
